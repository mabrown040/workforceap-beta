# Job Search Weekly Planner

Structure your job search for consistency and momentum.

## Weekly Goals

| Day | Activity | Target |
|-----|----------|--------|
| Mon | Apply to jobs | 3–5 applications |
| Tue | Research companies | 2–3 target employers |
| Wed | Apply to jobs | 3–5 applications |
| Thu | Network (LinkedIn, events) | 5 new connections |
| Fri | Follow up on applications | 3–5 follow-ups |
| Sat | Skill building / practice | 1–2 hours |
| Sun | Rest / plan next week | — |

## Daily Checklist

- [ ] Check job boards (Indeed, LinkedIn, company sites)
- [ ] Tailor resume/cover letter for each application
- [ ] Log applications in a tracker
- [ ] Respond to messages within 24 hours

## Weekly Review

- How many applications submitted?
- Any interviews scheduled?
- What's working? What isn't?
- Adjust strategy for next week.
